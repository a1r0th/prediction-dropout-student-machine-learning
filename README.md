# prediction-dropout-student-machine-learning
model machine learning untuk memprediksi prediksi dropout mahasiswa 
dengan 3 algoritma :
  -  Gradient Boosting
  -  XGBoost
  -  CatBoost

akurasi yang saya bisa dapatkan itu 85-86% untuk 3 algoritma di atas
saya harap dengan membagikan ini teman teman dapat mengavaluasi kode saya agar menjadi pelajaran bagi saya yang belajar
